2. Erhobene Daten

Firebase-Authentifizierung: Beim ersten Start der App wird ein anonymer Zugang über Firebase erstellt, bei dem eine pseudonyme Nutzer-ID (UID) erzeugt wird. Optional kann ein Login mit Google erfolgen. Dabei werden ggf. Name, E-Mail-Adresse und ID an Firebase übermittelt. Diese Daten werden zur Wiederherstellung des Spielstands oder zur Synchronisierung verwendet.

Google Play Spiele-Dienste: Wenn Sie sich mit einem Google-Konto anmelden, können über Google Play Games Informationen wie Nutzer-ID, Spielername, Erfolge und Cloud-Spielstände verarbeitet werden. Dazu können auch Profilbildung, Messung der Werbeleistung sowie Auswahl und Bereitstellung personalisierter Inhalte gehören.

Die Verarbeitung erfolgt gemäß den Datenschutzbestimmungen von Google und unterliegt den im CMP angezeigten Zwecken und Partnern.

Ihre Auswahl zu den Datenschutz-Einstellungen wird lokal auf Ihrem Gerät gespeichert und beim nächsten Start automatisch berücksichtigt. Sie können diese jederzeit über die Datenschutzeinstellungen in der App ändern.
